PHP 5.2 and earlier (Example Artical)

Functions are not first-class functions and can only be referenced by their name, directly or dynamically by a variable containing the name of the function.[66] User-defined functions can be created at any time without being prototyped.[66] Functions can be defined inside code blocks, permitting a run-time decision as to whether or not a function should be defined. Function calls must use parentheses,

with the exception of zero argument class constructor functions called with the PHP new operator, where parentheses are optional. PHP supports quasi-anonymous functions through the create_function() function, although they are not true anonymous functions because anonymous functions are nameless, but functions can only be referenced by name, or indirectly through a variable, in PHP.[66]
